# Project Structure

- `src/lunar_fno/models`: 1D and 2D FNO model definitions.
- `src/lunar_fno/data`: NPZ dataset adapters for both modalities.
- `src/lunar_fno/train`: train/eval entrypoints for waveform and spectrogram runs.
- `src/lunar_fno/utils`: reproducibility, metrics, and IO helpers.
- `configs`: default YAML config files.
- `scripts`: convenience shell wrappers for the train commands.
- `docs`: usage, data contract, and reproducibility notes.
